📋 ОПИСАНИЕ ПРОЕКТА
--------------------------------------------------------------------------------
Учёт оборудования в аудиториях — это информационная система для автоматизации
учёта оборудования, управления инвентарём и контроля перемещения техники между
аудиториями в образовательных учреждениях.

Система позволяет вести централизованный учёт оборудования, отслеживать его
местоположение, техническое состояние, оформлять заявки на ремонт и формировать
отчётную документацию.


🎯 ЦЕЛЬ СОЗДАНИЯ ИС
--------------------------------------------------------------------------------
Автоматизация учёта оборудования, управления инвентарём и контроля перемещения
техники между аудиториями в образовательном учреждении.


📌 НАЗНАЧЕНИЕ СИСТЕМЫ
--------------------------------------------------------------------------------
• Хранение информации об оборудовании (компьютеры, проекторы, интерактивные
  доски и т.д.)
• Учёт местоположения оборудования (аудитории, кабинеты)
• Назначение ответственных лиц за оборудование
• Контроль технического состояния и ремонта оборудования
• Формирование отчётной документации (инвентаризационные ведомости, акты
  списания)


🔑 ОСНОВНЫЕ ВОЗМОЖНОСТИ
--------------------------------------------------------------------------------

[Управление оборудованием]
  ✓ Добавление новых единиц оборудования с указанием характеристик
  ✓ Учёт текущего местоположения (аудитория)
  ✓ Назначение ответственного лица
  ✓ Обновление информации (изменение характеристик, статуса)
  ✓ Списание устаревшего или неисправного оборудования

[Справочники]
  • Типы оборудования (компьютеры, проекторы, МФУ и др.)
  • Аудитории (номер, этаж, здание, вместимость)
  • Ответственные лица (ФИО, должность, подразделение, контакты)

[Перемещение оборудования]
  ↔ Оформление перемещения между аудиториями
  📝 Фиксация причины перемещения
  📋 История перемещений каждой единицы оборудования

[Ремонт и обслуживание]
  🔧 Создание заявок на ремонт
  📊 Типы работ (ремонт, диагностика, обслуживание, замена комплектующих)
  📈 Статусы заявок (открыта, в работе, закрыта)
  🏷️ Статусы оборудования ("исправен", "в ремонте", "списан", "резерв")

[Отчётность]
  📄 Инвентаризационная ведомость по аудитории
  🔧 Отчёт по оборудованию в ремонте
  👤 Отчёт по ответственному лицу
  📊 Общий отчёт по всему оборудованию
  💾 Экспорт отчётов в формат CSV

🚀 УСТАНОВКА И ЗАПУСК
--------------------------------------------------------------------------------

1. Клонирование репозитория:
   git clone <URL-репозитория>
   cd equipment_accounting

2. Создание виртуального окружения:
   python -m venv .venv
   .venv\Scripts\activate          # Для Windows

3. Установка зависимостей:
   pip install -r requirements.txt

4. Настройка базы данных:
   - Установите PostgreSQL
   - Создайте базу данных
   - Отредактируйте параметры подключения в database/connection.py

5. Запуск приложения:
   python main.py
